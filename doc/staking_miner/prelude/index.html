Module staking_miner::prelude

source ·
Expand description

Types that we don’t fetch from a particular runtime and just assume that they are constant all of the place.

It is actually easy to convert the rest as well, but it’ll be a lot of noise in our codebase, needing to sprinkle any_runtime in a few extra places.

Re-exports

  • pub use pallet_election_provider_multi_phase as EPM;

Constants

  • DEFAULT_URI
    Default URI to connect to.
  • LOG_TARGET
    The logging target.

Traits

  • BlockT
    Something which fulfills the abstract idea of a Substrate block. It has types for Extrinsic pieces of information as well as a Header.
  • HeaderT
    Something which fulfills the abstract idea of a Substrate header. It has types for a Number, a Hash and a Hashing. It provides access to an extrinsics_root, state_root and parent_hash, as well as a digest and a block number.
